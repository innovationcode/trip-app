# Travel-app

- You can search restaurants, trip attractions on google map using this API

## Build using React stack and Rapid API for travel and weather

### Dependencies used

    - For styling app
        - @material-ui/core
        - @material-ui/icons
        - @material-ui/lab

    - For API fetching and google maps
        - axios
        - @react-google-maps/api
        - google-map-react
